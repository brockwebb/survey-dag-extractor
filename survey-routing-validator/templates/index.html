About This Tool

Purpose: Validate and correct survey routing logic extracted from PDF documents.

Workflow:

  1. Load survey from minimal JSON extraction
  2. Review each question systematically
  3. Define response options and routing rules
  4. Add terminal nodes as needed
  5. Export validated DAG in v1.1 format

Key Features:

  • Auto-generates predicate ASTs
  • Validates categorical encoding
  • Creates complete terminal nodes
  • Progress tracking and session persistence
